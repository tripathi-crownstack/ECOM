Hoisting
  • Hoisting refers to the initialization and declaration of variables at the top of the Scope.
  • Only var is declared as well as initialized but let and const are only declared but not initialized.They are known as temporal dead zone.
Closures
  • Functions along with its lexical environment forms a closure.
  • Every function in javascript has access to its scoped variables and its lexical environment (Environment in which the function is declared).
  • Important point to be noticed here is that if every functions remembers its lexical environment means the inner function knows its parent variable , in the the same way the parent variable also knows the lexical environment of its grand-parent's variables and so on.
